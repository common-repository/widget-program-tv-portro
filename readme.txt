A plugin who displays a TV program for certain channels. This widget brings the tv program to you and your visitors.

== Description ==
This widget gives the possibility of embedding the Widget Program TV Port.ro into any WordPress site.
Widget Program TV Port.ro is a plugin who displays the TV programm for 120 channels.
You can configure the display of the widget : colors,fonts,size,width,height and choose what Tv channels to show.
Also you may choose from a list of 11 predefined templates : "Now on TV","Sport on TV","Documentary","Cartoons" etc.
If you want a custom template, you may ask one at widget@port.ro or from widget.port.ro contact section.


This widget brings the tv program to you and your visitors.

== Installation ==

In order to use this plugin you have to create an account at http://widget.port.ro and configure some TV programs.
Each configured Tv program provides you a unique id/code who can be used to setup a "Widget Program TV Port.ro" plugin.

1. Upload `WidgetPortRo.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag&drop the plugin where do you want and provide the widget id. The widget id can be retrieved from http://widget.port.ro after you login and create a widget.
